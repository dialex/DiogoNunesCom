JColor is a Java library that offers you a fluent API to format terminal
messages with color. I extracted it from a university project and open-sourced
it.

It started as a small helper in 2011, was rebuilt in 2016, and modernised again
in 2020 with a fluent API, proper releases on Maven Central, and full
documentation.
